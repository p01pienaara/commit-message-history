This script scans the pubspec in a Flutter project to get all changes for its dependecies based on a given release tag range and compiles the list of commits in a commit log

This is the flow:

![image](https://github.com/p01pienaara/commit-message-history/assets/80837554/40704f19-4d3f-4ba7-bae6-06a3f3022aa8)


To run, clone the repo and run `main.py <old_version> <new_version>` using Python 3.11
